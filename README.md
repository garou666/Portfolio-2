# Single-File Portfolio Website

This is a personal portfolio website built from a single HTML file that contains all the necessary HTML, CSS, and JavaScript.

## Features

-   **All-in-One**: A complete website in one file for easy sharing and hosting.
-   **Responsive Design**: Adapts to screens of all sizes, from mobile phones to desktop computers.
-   **Dark Mode**: Includes a toggle for a dark color scheme, with the user's preference saved.
-   **Embedded Project Data**: Projects are managed within the JavaScript portion of the file.

## Tech Stack

-   **Frontend**: HTML5, CSS3, JavaScript (ES6+)

## How to Use

Simply open the `index.html` file in any modern web browser to view the portfolio.
